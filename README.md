## 🎯 About

A **stunning**, **performant**, and **fully responsive** portfolio website showcasing modern web development practices. Built with the latest technologies including **Next.js 16**, **React 19**, and **Tailwind CSS 4**, this portfolio features mesmerizing animations, a beautiful dark theme with gradient accents, and a seamless user experience.

## 🚀 Getting Started

### Prerequisites

- **Node.js** 18+ installed
- **npm** or **yarn** package manager

### Installation

```bash
# 1️⃣ Clone the repository
git clone https://github.com/your-username/portfolio.git
cd portfolio

# 2️⃣ Install dependencies
npm install

# 3️⃣ Start the development server
npm run dev

# 4️⃣ Open in browser
# Navigate to http://localhost:3000
```

## 🎨 Customization

### Theme Colors

Modify `app/globals.css` to customize the color palette:

```css
:root {
  --primary: #6366f1;      /* Indigo */
  --secondary: #8b5cf6;    /* Purple */
  --accent: #ec4899;       /* Pink */
  --background: #0f172a;   /* Dark Slate */
  --foreground: #f8fafc;   /* Light Gray */
}
```

### Key Customization Points

| Section | File | What to Update |
|---------|------|----------------|
| **Hero** | `components/sections/Hero.tsx` | Name, role titles, social links |
| **About** | `components/sections/About.tsx` | Bio, experience, achievements |
| **Skills** | `components/sections/Skills.tsx` | Your skillset |
| **Projects** | `components/sections/Portfolio.tsx` | Your projects |
| **Contact** | `components/sections/Contact.tsx` | Email, phone, location |
| **SEO** | `app/layout.tsx` | Title, description, keywords |

📚 See [CUSTOMIZATION.md](./CUSTOMIZATION.md) for detailed instructions.
